

# AI Humanizer – Polish Language Landing Page

A clean, minimal landing page for MVP testing of an AI text humanizer tool for Polish language.

## Sections

### 1. Navigation Bar
- Logo/brand name on the left
- Minimal nav links (How it works, Pricing)
- CTA button "Dołącz do listy" (Join waitlist)

### 2. Hero Section
- Bold headline communicating the core value: humanize AI-generated Polish text
- Supporting subheadline explaining the benefit (bypass AI detectors, natural-sounding Polish)
- Email input + "Join Waitlist" button
- Clean illustration or abstract visual element

### 3. How It Works (brief)
- 3-step visual flow: Paste text → AI humanizes → Get natural Polish output
- Simple icons with short descriptions

### 4. Pricing Plans
- 2-3 tier cards starting from $10
- Basic ($10), Pro, and Premium tiers with feature lists
- Each card has a "Join Waitlist" CTA (since payments aren't live yet)

### 5. Footer
- Simple footer with copyright and placeholder links

## Waitlist Functionality
- Email collection via a simple form (stored in local state for now, with a success toast notification)
- Can be upgraded to Supabase storage later when ready to collect real signups

## Design
- Clean white background with subtle gray accents
- Modern sans-serif typography
- Minimal use of color – one accent color for CTAs
- Smooth scroll between sections
- Fully responsive (mobile-first)
